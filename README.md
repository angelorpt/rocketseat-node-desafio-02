# Desafio 02 - Desafio: Trabalhando com Middlewares - Rocketseat

Projeto Desafio para testas os conceitos de middlewares em uma API Expres.js

<br>

## Instalação

1. No diretório do projeto

```bash
npm install
```

### Executando os Testes

```bash
npm test
```

## Executando o projeto

### Em desenvolvimento

```bash
nodemon
```

### Acessando o projeto:

```http
http://localhost:3000
```

## Rotas

| Verbo  | Recurso         |
| ------ | --------------- |
| GET    | /users          |
| GET    | /users/:id      |
| POST   | /users          |
| PATCH  | /users/:id/pro  |
| GET    | /todos          |
| GET    | /todos/:id      |
| POST   | /todos          |
| PUT    | /todos/:id      |
| PATCH  | /todos/:id/done |
| DELETE | /todos/:id      |
